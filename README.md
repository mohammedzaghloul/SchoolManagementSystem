# 🏫 نظام إدارة المدرسة (School Management System)

نظام متكامل لإدارة المدارس يربط بين الطلاب، المدرسين، وأولياء الأمور بواجهة مستخدم عصرية ونظام إشعارات لحظي.

## ✨ المميزات الرئيسية
- **نظام شات متطور:** تواصل مباشر بين الطلاب والمدرسين، وبين أولياء الأمور والمدرسين.
- **إشعارات لحظية:** تنبيهات فورية للرسائل الجديدة والأنشطة الأكاديمية (SignalR).
- **تصميم مظلم فاخر (Premium Dark Mode):** واجهة مريحة للعين بتصميم Indigo عصري.
- **تجاوب كامل (Responsive Design):** يعمل بسلاسة على كافة الشاشات (Mobile/Tablet/Desktop).
- **إدارة الحضور والغياب:** تتبع حضور الطلاب وتنبيه أولياء الأمور.
- **إدارة الواجبات والامتحانات:** رفع المهام الدراسية ومتابعة الدرجات.

## 🛠️ التكنولوجيا المستخدمة
- **Backend:** ASP.NET Core 10, Entity Framework Core, SignalR, MediatR.
- **Frontend:** Angular 19, Bootstrap 5, RxJS.
- **Database:** SQL Server (LocalDB).

## 🚀 كيفية التشغيل
لقد أضفت ملفاً بسيطاً لتشغيل المشروع بالكامل بضغطة واحدة:

1. تأكد من تثبيت **.NET SDK** و **Node.js**.
2. قم بتشغيل ملف `Run_Project.bat` الموجود في المجلد الرئيسي.
3. سيقوم الملف بتشغيل السيرفر (Backend) والواجهة (Frontend) تلقائياً.
